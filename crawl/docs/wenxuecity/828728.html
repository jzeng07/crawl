为了赶上傍晚在悉尼歌剧院的节目，吃完午饭后我们就赶了回来。尽管来去匆匆，还是对蓝山的美有了感性的认识。

悉尼歌剧院是悉尼的名片。到悉尼而没有去过悉尼歌剧院将是一个遗憾。下面的介绍来自于维基百科：

悉尼歌 剧 院 （ ：Sydney Opera House）位於 ，是20世紀最具特色的建筑之一，也是世界著名的表演艺术中心、悉尼市的标志性建筑。该剧院设计者为丹麦设计师 ，建设工作从1959开始，1973年大剧院正式落成。在2007年6月28日这栋建筑被 评为 。

悉尼歌剧院坐落在悉尼港的便利朗角（Bennelong Point），其特有的帆造型，加上作为背景的 ，与周围景物相映成趣。每天都有数以千計的遊客前來观赏这座建筑。其总面积为2.2公顷，由588个沉降于海平面以下25米的混凝土墩支撑。其中长183米，宽120米的建筑群占据了1.8公顷。整栋建筑建造在一块混凝土基座上，其入口开向南边，以97米宽的台阶同路面相连。这个建筑群可大体分为三个部分：最大的两组风帆沿南北方向排列，以略呈V字形的方式在南部收拢。它们分别以三面风帆面向北部的海湾，另一面则面向南面迎接观众和游客，风帆张开的“嘴”上则覆盖着茶色的玻璃幕墙。另一对小风帆位于它们的西南处，位于其中的乃是贝尼朗餐厅。整个建筑最高处为西面大厅的顶点，它距离海平面67米。

悉尼 是一个设计新颖的 ，它有一系列被称为 的大型预制混凝土构件，每一个构建都取自拥有相同半径的 ，它们构成了剧院的屋顶。尽管悉尼歌剧院德屋顶结构通常被称为“壳”，事实上它们在构造上严格的来说并不是“壳”，而是由肋骨状的预制混凝土构件支撑的 嵌板。整个建筑共由2,194个这样的嵌板，每个的重量大约是15吨，整个屋顶的重量为2万7千吨，它覆盖着1,056,006块由 哈格纳斯（Höganäs）制造的光滑的白色有人字纹路的瓷砖，从远处看这些“壳”显得非常洁白。这些瓷砖具有自动清洗的性质，不过还是会受到定期维护和更换。

音乐厅位于西面的一系列“壳”内，而歌剧院则在东面的“壳”内。“壳”的高度是由它内部所要求的高度决定的，从入口处的矮空间一直升高到位于最高处的台阶的座位区。一些次要的场馆位于音乐厅下面的混凝土基座内，是位于西面的“壳”的一部分。这个建筑拥有将近1000个房间，其建筑面积为8万8千平方米。整栋建筑的用电量相当于一个2万5千人的小城，总电线长度达645公里。

悉尼歌剧院主要由兩個主厅、一些小型剧院、演出厅以及其他附屬设施組成。兩个大厅均位於比较大的帆型結构内，小演出厅則位于底部的基座内。其中最大的主厅是音乐厅，最多可容纳2679人。设计的初衷是把这个最大的厅堂建造成為歌剧院，後來设计改动了，甚至已经完工的歌剧舞台被推倒重建。音乐厅内有一個巨大的管风琴，是由罗纳德·沙普（Ronald Sharp）于1969年至1979年制造的。号称是全世界最大的机械木連杆风琴，由10,500根风管組成。

主厅中较小的一个才是歌剧院。由于当初是將较大的主厅设计为歌剧院，小厅被认为不太适合做大型的歌剧演出，舞台相对较小而且給乐队的空間也不便于大型乐队演奏。

当晚的演出有5位艺术家表演著名歌剧的片断。其中一位男高音是韩国人，男中音是西班牙人，一位女高音是美国人。演出非常精彩。可以说是余音缈缈，绕梁三日。其中唱到“卡门”时全场观众一起唱起来，真是享受。我们邻座是两位衣着优雅的老两口。他们是悉尼本地人，住在乡下，每个月都会来歌剧院看一次表演。看歌剧已经成为他们生活中一个不可或缺的组成部分。

在去歌剧院的路上，有两个中国人在路边拉横幅，看样子是x轮功的人。我们不相信任何宗教，但也尊重他人的信仰。可是这俩个人听见我们在说中国话，就跟在我们旁边不停地叨叨，我们开始很和气地告诉他们我们不感兴趣，但他们仍然喋喋不休，说个不停，直到我们大声喊道：“STOP！ SHUT UP！”， 他们才离开。真不明白他们为什么要这么做。有意思吗？ 弄的我们心里不痛快了好一阵。

从悉尼歌剧院出来，正值夕阳西下，远眺悉尼海港大桥，近观悉尼歌剧院，水面波光粼粼，微风中传来无数年轻人的歌手和欢笑声，充满了一种让人心旷神怡的美。

在夕阳下，我们慢慢地走到悉尼环形码头。悉尼环形码头位于便利朗角（Bennelong Point）与岩石区（The Rocks）之间的悉尼湾悉尼商业中心区的北部边缘，它也是悉尼市地方政府区域的一部分。环形码头由海滨小径、行人购物中心、公园、餐厅和咖啡座组成，并设有火车站和多个渡轮码头。环形码头呈半环形，原称半环形码头，后来才简化成今日的名称。

环形码头历史上是澳洲原住民的发源地，悉尼是欧洲的首个 ，而悉尼湾則是英国海军上将亚瑟 船長所率領的 第一舰队在抵达澳洲后最先上岸的地点。环形码头是悉尼主要的交通枢纽中心，设有一个大型的渡轮，巴士与铁路转运站。Cahill高速公路是环形码头的重要设施，从东面通往环形码头火车站，延伸到悉尼海湾大桥。环形码头的渡轮码头有5条渡轮航线，是所有悉尼港与帕拉马塔河公众渡轮航线的终点。

岩石区建筑

环形码头旁边就是大名鼎鼎的岩石区（The Rocks）。悉尼早期的囚犯生活在岩石区，这个地区有着杂乱的鹅卵石街道和死胡同，距离环形码头（Circular Quay）仅五分钟路程。只需走下海滨，就可以看到砂岩台地和小屋，以及一些悉尼最古老的酒吧。走在这里似乎街上每一块鹅卵石，街角每一栋古老的建筑都隐藏着一段不知名的早期移民的故事。当我们今天坐在酒吧欣赏现场音乐或品尝当地酿制的啤酒，想一想十九世纪的水手、士兵，街头恶棍和码头工人们曾在这里举杯庆祝或是借酒浇愁，那会是一种什么样的情景？ 从在这些人的故事中，今天的年轻人能体会出他们生活里的激情，艰辛和苦难吗？这片历史悠久的区域现在建有博物馆、艺术画廊、热闹的周末市场和可以欣赏到海港风景的酒店。 在1月26日的澳大利亚日（ Australia Day）这里有乐队现场演出，在新年夜可以欣赏悉尼海港大桥上空绽放的焰火。

岩石区的历史建筑

岩石区一直延伸到悉尼海港大桥（Sydney Harbour Bridge）壮观的拱形桥身。悉尼海港大桥号称是世界第一单孔拱桥，是早期悉尼的代表建筑，它像一道横贯海湾的长虹，巍峨俊秀，气势磅礴，与举世闻名的悉尼歌剧院隔海相望，成为悉尼的象征。悉尼海港大桥有许多重要的意义，她是连接港口南北两岸的重要桥梁、是悉尼歌剧院明信片的完美背景、在距离水面147米的高处遥望悉尼歌剧院，这个角度绝对独一无二，也是摄取港口全景的绝佳地点。以下的数据和说明摘自于百度百科。

悉尼海港大桥，被当地居民称为“衣服架”，从"怀胎"到"出世"，前后花费了100多年。在经过了40多年的酝酿之后，1857年，悉尼工程师彼得·翰德逊绘成了第一张设计图，其后经过反复修改，到1923年才根据督建铁路桥的总工程师卜莱费博士的蓝图进行招标，由英国一家工程公司中标承建。悉尼海港大桥动土的官方仪式于1923年7月28日进行，1932年3月19日竣工通车，历时8年多。

岩石区的教堂

悉尼海港大桥

这座大桥整个工程的耗资达1350万澳元(约合690万美元)。这座桥由大约600万颗铆钉固定，桥拱的跨度为503米，最高处距离海面134米。全部用钢量为5.28万吨，铆钉数是600万个，最大铆钉重量3.5公斤，用水泥9.5万立方米，桥塔、桥墩用花岗石1.7万立方米，建桥用油漆27.2万升，从这些数字足可见铁桥工程的雄伟浩大。在30年代的条件下，能在大海上凌空架桥，实为罕见。整个悉尼海港大桥桥身长度（包括引桥）1149米，从海面到桥面高58.5米，

从海面到桥顶高达134米，万吨巨轮可以从桥下通过。桥面宽49米，可通行各种汽车，中间铺设有双轨铁路，两侧人行道各宽3米。原来还铺设有轨电车车轨两条，后因交通拥挤把它拆除，划出8条汽车道。悉尼海港大桥的最大特点是拱架，其拱架跨度为503米，而且是单孔拱形，这是世界上少见的。大桥的钢架头搭在两个巨大的钢筋水泥桥墩上，桥墩高12米。两个桥墩上还各建有一座塔，塔高95米，全部用花岗岩建造。2012年悉尼海港大桥的交通完全由电脑控制。桥上还有巡逻车巡逻，随时处理各种情况，使大桥始终保持畅通无阻。建成的悉尼海港大桥长502.9公尺，宽48.8公尺，有8个车道，2条铁轨，1条自行车道及1条人行道。

无论乘飞机还是乘海轮到达悉尼，最先映入眼帘的就是这座全世界最宽的大铁桥。在斜阳的金晖中，拱桥映衬着变幻的夕阳，由金黄而姹紫，由姹紫而暗褐，海面上的桥影刚刚褪淡，立刻有闪亮起斑斓的光带。大桥钢架、栏杆上的璀璨的千万盏灯光，远望恍惚是在弓背和弓弦上镶嵌的无数钻石，在黑丝绒般的夜空下，仪态万千；再衬上波平如镜的海湾中五颜六色的霓虹灯倒影，平添几分神秘。桥上往来的车辆就像是一排排整队爬行的蚂蚁，十分壮观。每到晚上，只见桥的钢架和栏杆上，都亮起了电灯，这些灯在深黑的夜空下，在神秘的海湾中像星星一样眨着眼。澳大利亚人形容悉尼海港大桥的造型像一个"老式的大衣架"，并把它誉为悉尼的象征。由于悉尼海港大桥和悉尼歌剧院相邻，人们将歌剧院和大桥联成一体欣赏时，雄伟和婀娜、深色和浅色、直线和曲线构成了一幅反差强烈又协调一体的美丽图画，真是相映成辉。

仔细看，你可以看到有人在攀爬大桥

攀爬悉尼海港大桥(Harbour Bridge)是最受欢迎的旅游项目之一。澳大利亚人爱玩，所以才会想出了要攀爬这个绰号为“大衣架”的世界上最大的拱桥，这也是世界上唯一允许游客爬到拱桥顶端的大桥。从1998年开始，悉尼大桥开放给公众攀爬，整个攀桥过程非常安全。在登桥之前，必须通过肺活量测验、酒精测验以及通过金属探测器，还必须请你在健康状况证明和合约表上签名才可以获准攀爬。开始前全部的人都会拿到一套攀登服。另外，如果是晚间攀登还会有轻型的攀桥专用灯。整个攀桥旅程历时3小时。成功攀上悉尼大桥顶端的游客都会收到一份纪念照片以及成功攀登大桥的证书。如果退回20年，我们不玩别的也会去爬这座大桥。

在这个温暖的夏日周末的傍晚，微微的海风习习扑面而来，落日的余晖映照着波光粼粼的水面，无数年轻人涌进从悉尼歌剧院，到环形码头，到岩石区海湾沿线上百家酒吧和餐厅，他们的歌声，欢笑声随着海波荡漾。我们坐在悉尼歌剧院对面岩石区的临湾餐厅，品着澳洲的红酒，尝着意大利美食，看着水面上船来船往，听着年轻人的欢歌笑语，还有那被落日的余晖逐渐染红的悉尼歌剧院，渐渐的有些醉了…
