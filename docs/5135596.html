《外交政策》（Foreign Policy）网站12日刊登题为《为川普投票就是为中国投票》（A Vote for Trump Is a Vote for China）的文章。作者艾萨克•斯通•菲什（Isaac Stone Fish）是该刊亚洲区编辑，曾在中国居住7年。

作者在文中称，川普在竞选总统的过程中收获了许多外号，但其中有一个极具讽刺意味——“中国卧底”。这个称呼最先来源于美国华文媒体 。为什么这么称呼呢？因为川普试图保护美国利益的措施其实会让中国获益。

川普在竞选活动中一直炮轰中国，比如说中国操纵货币、窃取美国人的工作等。但有专家表示，川普若当选总统将会加固中国共产党的统治及中国在世界上的地位。

几十年来，川普都主张对日关系降温。日本在美国外交政策中扮演重要角色，是美国在亚洲安全战略的关键。日本有超过5万的美国驻军，在该地区起到了遏制中国和朝鲜的作用。川普3月25日在接受《纽约时报》采访时表示：如果日本和韩国不支付更多维持费用，美国就应撤走驻军并重新思考与日本的安全联盟。

从日本或韩国撤走驻军将会削弱美国在该地区的地位，并“令中国得逞，”日本神奈川大学国际关系副教授佐桥亮（Ryo Sahashi）说。这会极大地减损华盛顿在南中国海压制北京的能力。

还有人担心，川普对国际组织的态度会为中国权力的增加创造条件。川普在最近的演讲中谴责联合国组织脆弱无能，称其“不是民主和自由的朋友，也不是美国的朋友”。但中国主席习近平却希望加深与国际组织的关系。去年他在联合国的首次演说中承诺设10亿美元的联合国和平发展基金。如果美国在联合国退缩，中国将取得更大的影响力。

川普还说要给中国出口商品加45%的关税，但这会让美国损失比中国更大。布兰迪斯大学经济学者皮特•佩特里（Peter Petri）估算，这会导致美国商品贸易逆差增至670亿美元。贸易战争会“影响美国利率和证券市场，海关税的‘保护’其实会带来更大伤害，”战略与国际关系中国中国经济专家斯考特•肯尼迪（Scott Kennedy）说。

川普这种制裁固然能损害中国，但中国经济发展已经不再主要依靠出口，而是依靠国内消费市场、投资及政府支出。2006年，贸易占中国GDP总额的65%，到了2014年，该比例下降至42%。川普若使出口美国不再具有吸引力，其实是帮中国建设更稳定的经济。

和川普一样，习近平也喜欢“复兴”这个说法。但很讽刺的是，这两个有可能同期当政的人，都是为了实现中国的复兴。
